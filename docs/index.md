# gmalg 用户指南

欢迎来到 gmalg! 一个纯 Python 实现的国密算法库, 包含 SM9 在内的多个国密算法, 开箱即用!

## 安装

```bat
pip install gmalg
```

## 包含的算法

- SM2 椭圆曲线公钥密码算法
    - 签名验签
    - 密钥交换
    - 加密解密
- SM3 密码杂凑算法
- SM4 分组密码算法
- SM9 标识密码算法
    - 签名验签
    - 密钥交换
    - 密钥封装
    - 加密解密
- ZUC 序列密码算法
